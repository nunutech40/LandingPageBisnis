# Strategi Dominasi SEO & Checklist "Page One"

Dokumen ini adalah panduan taktis untuk memastikan Landing Page kita (dan klien) memenuhi standar ranking Google Halaman 1.

## 1. Filosofi SEO Teknikal ("The Google Language")
Google bukan manusia. Dia adalah robot yang membaca kode. Agar menang, kita harus menyajikan kode yang paling mudah dimengerti oleh robot tersebut.

### Fokus Utama:
1.  **Crawlability:** Memastikan Google bisa membaca semua halaman.
2.  **Indexability:** Memastikan halaman layak masuk database Google.
3.  **Rankability:** Memastikan konten cukup berwibawa untuk tampil di atas.

## 2. Checklist "Page One Ready" (Standar Audit)

Gunakan checklist ini sebelum meluncurkan landing page apapun.

### A. Technical SEO (Wajib 100%)
- [ ] **Schema Markup Valid:** Tidak ada error di [Rich Results Test](https://search.google.com/test/rich-results).
    - [ ] `LocalBusiness` (Nama, Alamat, Telp, Jam Buka).
    - [ ] `Service` (Nama Jasa, Area).
    - [ ] `FAQPage` (Pertanyaan umum).
- [ ] **Core Web Vitals Lulus:**
    - [ ] LCP < 2.5 detik.
    - [ ] CLS < 0.1.
    - [ ] INP < 200ms.
- [ ] **HTTPS/SSL:** Gembok hijau aktif.
- [ ] **Robots.txt & Sitemap.xml:** Ada dan tidak memblokir bot Google.
- [ ] **Canonical Tag:** Mencegah isu konten duplikat (`<link rel="canonical" ... />`).
- [ ] **Mobile Friendly:** Lulus tes [Mobile-Friendly Google](https://search.google.com/test/mobile-friendly).

### B. On-Page SEO (Konten & Struktur)
- [ ] **Title Tag:** Mengandung Keyword Utama + CTA. (Maks 60 karakter).
    - *Contoh:* "Jasa Bangun Rumah Mewah Jakarta | Garansi 10 Tahun"
- [ ] **Meta Description:** Mengandung Keyword + Benefit. (Maks 160 karakter).
- [ ] **Heading Structure (H1-H6):**
    - [ ] Hanya SATU tag `<h1>`.
    - [ ] Keyword turunan di `<h2>` dan `<h3>`.
- [ ] **Keyword Density:** Natural (jgn spam), fokus di paragraf pertama dan terakhir.
- [ ] **Alt Text Gambar:** Semua gambar punya deskripsi keyword (e.g., `alt="kontraktor kolam renang jaksel"`).
- [ ] **Internal Linking:** Link ke halaman relevan lain (jika ada).

### C. Off-Page & Authority (Sinyal Luar)
- [ ] **Google My Business (GMB):** Terverifikasi dan 100% lengkap.
- [ ] **Reviews:** Minimal 5 review bintang 5 di GMB (Embedded di web).
- [ ] **Social Signals:** Link ke profil IG/FB/LinkedIn bisnis aktif.

## 3. Strategi Konten "Topical Authority"
Jangan cuma jualan. Jawab semua pertanyaan user.

**Struktur Konten Landing Page:**
1.  **Problem Awareness:** "Pusing atap bocor terus?" (Target Pain Point)
2.  **Agitation:** "Kalau dibiarkan, plafon bisa rubuh dan biaya bengkak."
3.  **Solution (Your Service):** "Kami spesialis anti-bocor..."
4.  **Authority (Proof):** "Sudah mengerjakan 500+ rumah."
5.  **FAQ (SEO Goldmine):** Jawab pertanyaan 5W+1H di sini untuk nangkep keyword panjang (*long-tail*).
    - *Contoh:* "Berapa biaya per meter?", "Apakah bergaransi?", "Berapa lama pengerjaan?"

## 4. Tools Verifikasi
Gunakan tools ini untuk mencentang checklist di atas:
1.  **Google PageSpeed Insights:** Cek performa.
2.  **Schema.org Validator:** Cek struktur JSON-LD.
3.  **Ahrefs/Ubersuggest (Free Tools):** Cek densitas keyword sederhana.
4.  **Screaming Frog (Free ver):** Cek broken link.
